# Calculator-App
A Java console-based calculator with basic arithmetic operations, history tracking, and the ability to use previous results.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division, modulus, exponentiation
- **Unary Operations**: Square root, absolute value, base-10 logarithm
- **History**: Automatically saves all calculations
- **Previous Answer**: Option to use the result from the last calculation
- **Input Validation**: Handles invalid inputs and division by zero

## Operations

| Operator | Description |
|----------|-------------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulus |
| `^` | Exponentiation |
| `r` | Square root |
| `A` | Absolute value |
| `L` | Base-10 logarithm |
